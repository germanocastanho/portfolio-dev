# 💻 Main Technologies

- [**Vite**](https://vitejs.dev/) - Fast build tool for modern web development.
- [**TypeScript**](https://www.typescriptlang.org/) - Strongly typed language for JavaScript.
- [**React**](https://reactjs.org/) - Library for building user interfaces.
- [**Shadcn/ui**](https://ui.shadcn.com/) - Component library for building modern UIs.
- [**Tailwind CSS**](https://tailwindcss.com/) - Utility-first CSS framework.

# ✅ Prerequisites

- **Node.js** & **NPM**, preferably installed via NVM.
- **NVM**, available at its [**GitHub Repository**](https://github.com/nvm-sh/nvm).

# 🛠️ Local Installation

```bash
# Clone the repository
git clone https://github.com/germanocastanho/portfolio-dev.git

# Navigate to the directory
cd portfolio-dev

# Install dependencies
npm install

# Start the development server
npm run dev
```

# 📦 Manual Deployment

```bash
# Create a production build
npm run build
```
